# Modelling Voter Responses to Pre-electoral Coalitions
This is the code accompanying the final project of [2026 Topics in Computational Social Choice project](https://theo.delemazure.fr/teaching/comsoc-26).

## Installation
The required python packages are `jupyter pref_voting tabulate matplotlib`.

Once they are installed one can start the jupyter notebook
```
$ jupyter notebook
```
and open the provided `.ipynb` file.
